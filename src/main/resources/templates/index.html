Sobre o Sistema

O Mini ERP é um sistema completo de gestão empresarial que permite:

  • Gestão de Clientes: Cadastrar, editar e gerenciar informações de clientes
  • Controle de Produtos: Manter catálogo de produtos com preços e estoque
  • Processamento de Pedidos: Criar e acompanhar pedidos de vendas

Use o menu lateral para navegar entre as funcionalidades do sistema.
